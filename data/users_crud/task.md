Напиши в  директории users_crud на python (используй uv) fastapi-проект с CRUD пользователей, используй REST.

По каждому пользователю:

id — bigint, autoincrement
first_name — varchar(200), not null
last_name — varchar(200), not null
email — varchar(255), not null

База данных SQLite, SQLAlchemy Core если она поддерживает SQLite. Используй асинхронный код для доступа к БД и в FastAPI. Используй типизированный код.

Работу с БД вынеси в слой репозитория. Для миграций используй Alembic.

Используй ruff в качестве линтера и убедись, что код успешно проходит его проверки.

Убедись, что все CRUD сервисы работают. Напиши также тесты (используй pytest) на веб-ручки, пусть эти тесты работают с тестовой БД users_db_test.
